# StayFinder

StayFinder is a simple frontend web application that helps users find the best rooms and mess services near their location.  
It is designed with a clean and user-friendly interface to make searching for accommodation easier.


## Features

- Find nearby rooms
- Find nearby mess services
- Simple and easy-to-use interface
- Responsive frontend design


## Technologies Used

- HTML
- CSS
- JavaScript
